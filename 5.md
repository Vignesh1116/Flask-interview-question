Flask Advanced Concepts

41. What are Blueprints?

Blueprints organize large Flask applications into modules.

42. What are Flask extensions?

Extensions add additional features such as:

authentication

database support

caching

43. What is Flask CLI?

Flask CLI allows running Flask commands from the terminal.

44. What is static folder in Flask?

The static folder stores files like CSS, JS, and images.

45. What is template folder?

The templates folder stores HTML files.

Flask Deployment
46. What is WSGI?

Web Server Gateway Interface connects web servers with Python applications.

47. What is Gunicorn?

Gunicorn is a production WSGI server used to deploy Flask apps.

48. What is Nginx used for?

Nginx acts as a reverse proxy for Flask applications.

49. What is Docker deployment?

Docker allows packaging Flask applications into containers.

50. What is cloud deployment?

Flask apps can be deployed on:

Amazon Web Services

Google Cloud

Microsoft Azure
