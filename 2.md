11. What is dynamic routing?

Dynamic routing allows URLs to include variables.

Example concept:

/user/<username>

12. What is URL building in Flask?

Flask uses url_for() to generate URLs dynamically.

Benefits:

Avoid hardcoding URLs

Easier maintenance

13. What HTTP methods does Flask support?

Flask supports common HTTP methods:

GET

POST

PUT

DELETE

PATCH

14. What is request object?

The request object contains data sent by the client.

Example data:

form data

headers

query parameters

15. What is response object?

The response object contains data returned to the client.

Flask Templates
16. What is Jinja2?

Jinja is the template engine used by Flask.

It allows embedding Python-like expressions in HTML.

17. What are templates in Flask?

Templates are HTML files used to render dynamic content.

They are stored in the templates folder.

18. What is template inheritance?

Template inheritance allows reusing common layouts across multiple pages.

Example concept:

base template

child templates

19. What are template variables?

Template variables allow passing dynamic data from Python to HTML.

20. What are template filters?

Filters modify variables inside templates.

Example filters:

upper

lower

length
