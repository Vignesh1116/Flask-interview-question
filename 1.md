1. What is Flask?

Flask is a lightweight Python web framework used to build web applications and REST APIs.

2. Who created Flask?

Flask was created by Armin Ronacher.

3. Why is Flask called a micro framework?

Flask is called a micro framework because it provides minimal features and allows developers to add extensions as needed.

4. What are the main features of Flask?

Lightweight

Flexible

Easy routing

Built-in development server

Template engine support

5. What is routing in Flask?

Routing is the process of mapping URLs to Python functions.

Example concept:

/home → home() function

6. What is a Flask application object?

The Flask application object is the main entry point of the application.

Example concept:

app = Flask(__name__)

7. What is __name__ used for in Flask?

__name__ tells Flask where the application is located so it can load resources like templates and static files.

8. What is the development server in Flask?

Flask includes a built-in development server used for testing applications during development.

9. What is a view function?

A view function handles HTTP requests and returns responses.

10. What is a Flask route decorator?

The route decorator connects a URL with a function.

Example concept:

@app.route("/")
